# 📌 Linear Regression Project (Python)

## 📖 Overview

This project demonstrates the implementation of **Linear Regression** using Python.  
The model predicts a continuous value (e.g., house price) based on input features (e.g., house size).

The project includes:
- Model training using Scikit-learn
- Data handling using NumPy
- Data visualization using Matplotlib
- Model evaluation using MSE and R² score

---

## 🧠 What is Linear Regression?

Linear Regression is a supervised machine learning algorithm used to predict continuous values using the equation:

y = wx + b

Where:
- w = slope (weight)
- b = intercept (bias)
- x = input feature
- y = predicted output

The goal is to minimize the cost function (Mean Squared Error).

## 📈 Cost Function

The cost function used is Mean Squared Error (MSE):

J(w, b) = (1 / 2n) * Σ(y_actual - y_predicted)^2

The objective is to minimize this cost.

---

## 🧪 Evaluation Metrics

- MSE (Mean Squared Error) → Measures average squared error
- RMSE (Root Mean Squared Error) → Square root of MSE
- R² Score → Measures goodness of fit
